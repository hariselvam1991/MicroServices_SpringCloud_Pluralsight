
	# Consume configuration values from our code
1. Create s Spring Starter project
2. Add application property file
3. Crete controller with annotations
4. Return values derived from properties
5. Experiment with different name or profiles
6. Make sure the config-server app is running
7. Changing spring.profiles.active=default to qa will bring qa properties from the config server
